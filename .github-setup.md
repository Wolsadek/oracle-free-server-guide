## 2️⃣ Criar Repositório no GitHub

1. Acesse https://github.com/new
2. Preencha:
   - **Repository name**: `oracle-free-server-guide`
   - **Description**: `🚀 Guia completo para configurar e monitorar servidor Oracle Cloud Always Free Tier com Docker, monitoring, backups e muito mais!`
   - **Visibility**: ✅ Public
   - **Initialize**: ❌ NÃO adicionar README, .gitignore ou license (já temos!)

3. Clique em **Create repository**

## 3️⃣ Conectar e Push

```bash
# Adicionar remote (SUBSTITUA SEU_USUARIO!)
git remote add origin https://github.com/SEU_USUARIO/oracle-free-server-guide.git

# Verificar
git remote -v

# Renomear branch para main (se necessário)
git branch -M main

# Push
git push -u origin main
```
