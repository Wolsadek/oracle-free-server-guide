# Contribuindo

Obrigado por considerar contribuir com este projeto! Toda ajuda é bem-vinda.

## Como Contribuir

### Reportar Bugs

Se encontrar um bug, abra uma issue incluindo:
- Descrição clara do problema
- Passos para reproduzir
- Comportamento esperado vs atual
- Screenshots (se aplicável)
- Sistema operacional e versão

### Sugerir Melhorias

Tem uma ideia para melhorar o guia? Abra uma issue com:
- Descrição clara da melhoria
- Por que seria útil
- Exemplos de uso (se aplicável)

### Pull Requests

1. Fork o projeto
2. Crie uma branch para sua feature (`git checkout -b feature/MinhaFeature`)
3. Commit suas mudanças (`git commit -m 'Add: minha nova feature'`)
4. Push para a branch (`git push origin feature/MinhaFeature`)
5. Abra um Pull Request

### Diretrizes para Código

- Use bash script seguindo as convenções do projeto
- Adicione comentários explicativos
- Teste seus scripts antes de submeter
- Mantenha o estilo de código consistente
- Atualize a documentação se necessário

### Diretrizes para Documentação

- Escreva em português claro e direto
- Use exemplos práticos
- Adicione screenshots quando útil
- Mantenha o formato markdown consistente
- Verifique a ortografia

## Tipos de Contribuição

### Scripts

- Novos scripts de monitoramento
- Melhorias nos scripts existentes
- Scripts para outros serviços (Docker, databases, etc)

### Documentação

- Correções de erros
- Novos tutoriais
- Traduções
- Melhorias de clareza

### Exemplos

- Casos de uso reais
- Configurações específicas
- Integrações com outras ferramentas

## Código de Conduta

- Seja respeitoso e construtivo
- Aceite críticas construtivas
- Foque no que é melhor para a comunidade
- Seja paciente com iniciantes

## Dúvidas?

Abra uma issue com a tag `question` ou entre em contato.

Obrigado! 🚀
